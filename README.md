# Kairos FX

A trading dashboard built with Next.js — charts, strategy backtesting, trade tracking, and analysis tools.

## Features

- **Charts** — live and historical price charts
- **Trades** — log and track your trades
- **Backtest** — test strategies against historical data
- **Strategy** — build and manage trading strategies
- **Analysis** — performance analytics

## Setup

**Requirements:** Node.js 18+

```bash
git clone https://github.com/brooksbilly246-gif/kairos-fx.git
cd kairos-fx
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Use with NEXUS

Kairos FX runs on port 3000 by default and embeds directly into the [NEXUS](https://github.com/brooksbilly246-gif/nexus) Kairos panel. Set `KAIROS_URL=http://localhost:3000` in your Nexus `.env`.
