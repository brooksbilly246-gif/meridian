# Meridian

A trading dashboard built with Next.js — charts, strategy backtesting, trade tracking, and analysis tools.

## Features

- **Charts** — live and historical price charts
- **Trades** — log and track your trades
- **Backtest** — test strategies against historical data with automated suggestions
- **Strategy** — London Session Breakout engine with risk management
- **Analysis** — performance analytics

## Setup

**Requirements:** Node.js 18+

```bash
git clone https://github.com/brooksbilly246-gif/meridian.git
cd meridian
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
